ExTrack: The Expense Tracking Application

×

This is a single page application to interact with the ExTrack (Expense Tracking) application through its REST API.

This View is available for any standard user. Users maybe created on the fly by anybody. However, to access a more complete view and to be able to list all the user, you need to login as an administrator.

Administrator accounts are created upon request. An administrator can delete an user from the application, subsequently deleting all the user's related records. So be careful when doing so. An admin account is provided for testing purpose.
